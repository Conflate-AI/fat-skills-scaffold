# Research Process State Machine

Every research initiative follows this sequence:

```
QUESTION ──► SEARCH ──► EXTRACT ──► SYNTHESIZE ──► VALIDATE ──► PUBLISH
  │              │          │            │             │           │
  │              │          │            │             │           └─ share findings, archive sources
  │              │          │            │             └─ cross-reference, check for bias
  │              │          │            └─ connect ideas, identify patterns, build knowledge
  │              │          └─ highlight, annotate, organize into structured notes
  │              └─ find sources, assess credibility, screen for relevance
  └─ frame the question, define scope, identify methodology
```

## Transition guards

| Transition | Guard |
|---|---|
| QUESTION → SEARCH | Research question is specific and answerable. Scope defined. Methodology chosen. |
| SEARCH → EXTRACT | Sufficient sources found (minimum depends on depth: quick=5, deep=20+). Source credibility assessed. |
| EXTRACT → SYNTHESIZE | Key insights extracted from all sources. Notes organized and linked. |
| SYNTHESIZE → VALIDATE | Synthesis complete. No major gaps in coverage. Contradictions identified and noted. |
| VALIDATE → PUBLISH | Claims cross-referenced. Bias checked. Evidence quality assessed. |

## SEARCH detail

1. **Keyword strategy** — primary terms, synonyms, related concepts, Boolean operators
2. **Source diversity** — academic, industry, primary data, expert opinion
3. **Screening** — title/abstract screen, then full-text screen for relevance
4. **Tracking** — maintain a source log with citations, access dates, and relevance ratings

## EXTRACT detail

1. **Structured notes** — consistent format per source: key findings, methodology, limitations, quotes
2. **Progressive summarization** — highlight → bold key points → write 1-paragraph summary
3. **Cross-linking** — link related notes, tag by theme
4. **Source tracking** — every note links back to its source with page/section references

## SYNTHESIZE detail

1. **Pattern identification** — what themes emerge across sources?
2. **Contradiction mapping** — where do sources disagree? Why?
3. **Gap identification** — what questions remain unanswered?
4. **Knowledge building** — create synthesis notes that connect individual findings into a coherent narrative
