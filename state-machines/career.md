# Career Development Process State Machine

Every career initiative follows this sequence:

```
ASSESS ──► PLAN ──► ACT ──► REFLECT ──► ADJUST
  │           │        │         │           │
  │           │        │         │           └─ course-correct based on reflection
  │           │        │         └─ review progress, gather feedback, celebrate wins
  │           │        └─ execute learning, applications, networking, interviews
  │           └─ set goals, identify gaps, build timeline
  └─ self-assessment, skills inventory, values clarification
```

## Transition guards

| Transition | Guard |
|---|---|
| ASSESS → PLAN | Skills inventory complete. Values and priorities clarified. Target direction identified. |
| PLAN → ACT | Goals are SMART. Timeline is realistic. Resources identified. |
| ACT → REFLECT | At least 2 weeks of consistent action. Milestones to review. |
| REFLECT → ADJUST | Progress honestly assessed. Feedback collected from trusted sources. |

## ASSESS detail

1. **Skills inventory** — what you're good at, what you enjoy, what the market values
2. **Values clarification** — what matters most (impact, income, growth, autonomy, balance)
3. **Career timeline** — map past roles, inflection points, and patterns
4. **360 input** — ask colleagues, mentors, and managers for honest feedback

## PLAN detail

1. **Goal setting** — SMART goals for 3-month, 6-month, 1-year horizons
2. **Gap analysis** — skills, experience, and credentials between current and target state
3. **Learning plan** — courses, books, projects, and mentorship with priorities and timelines
4. **Network strategy** — who to meet, where to find them, how to add value

## ACT detail

1. **Skill building** — follow the learning plan, track hours and outcomes
2. **Visibility** — share learnings, contribute publicly, build professional presence
3. **Networking** — regular outreach, informational interviews, community participation
4. **Applications** — tailored resumes, custom cover letters, interview prep per opportunity
